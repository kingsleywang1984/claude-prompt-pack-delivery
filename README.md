## Contents

50 hand-engineered prompts across 5 categories:

- STUDY — textbook compression, Anki generation, essay critique, paper triage
- CODE — senior code review, bug post-mortems, refactor justification, API design
- WRITE — voice-preserving editing, AI-smell removal, argument strengthening
- THINK — pre-mortems, calibration checks, opportunity-cost analysis
- SHIP — MVP scope-cutting, launch tweets, user interview scripts, retrospectives

Each prompt comes with a "why this works" annotation explaining the technique so you can adapt it.
